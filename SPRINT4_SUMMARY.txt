KEY FEATURES:
-------------

1. TAB DEDUPLICATION (Lines 505-530 in popup.js)
   - Prevents duplicate URLs during session restore
   - Uses Set data structure for fast lookups
   - Logs skipped duplicates to console

2. STATS DASHBOARD (Lines 994-1104 in popup.js)
   - Total sessions, tabs, and groups
   - Storage usage in KB
   - Oldest/newest session dates
   - Top 10 domains with counts
   - Real-time calculation on tab switch

3. BULK OPERATIONS (Lines 1106-1259 in popup.js)
   a. Close All Ungrouped Tabs
      - Skips pinned tabs for safety
   
   b. Remove Duplicate Tabs
      - Keeps first occurrence of each URL
   
   c. Close Old Tabs
      - User-defined days threshold (1-365)
      - Uses lastAccessed property
      - Confirmation dialog
   
   d. Group by Domain
      - Creates new group for matching tabs
      - Auto-names group with domain
